01. How to Use

Integration is seamless. Swap your default NTP server address with ours. We recommend configuring a standard pool as a fallback.

  • check

    Standard Protocol

    Uses standard UDP port 123.

  • check

    Anycast Routing

    Connect to the nearest node automatically.

ntp.conf 
# Use smoothtime pool
server europe-central.smoothtime.io iburst

# Fallback pool
pool pool.ntp.org iburst
